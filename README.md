# Eterno Training Club

Sitio web profesional y responsive para Eterno Training Club, gimnasio en Cabo San Lucas.

## Editar antes de publicar

- Reemplazar `52XXXXXXXXXX` en `script.js` por el numero real de WhatsApp.
- Confirmar telefono, correo, Instagram y direccion en `index.html`.
- Validar vigencia de precios antes de publicarlos.
- Conectar el formulario a un servicio real si no se quiere usar WhatsApp.
- Reemplazar enlaces legales de Aviso de privacidad y Terminos y condiciones.

## Publicar en GitHub y Vercel

1. Crea un repositorio en GitHub.
2. Sube todo el contenido de esta carpeta.
3. En Vercel, importa el repositorio.
4. Configura el proyecto como sitio estatico. No requiere build command.
5. Usa `index.html` como entrada principal.

## Estructura

- `index.html`: contenido y SEO.
- `styles.css`: diseno responsive.
- `script.js`: menu, filtros, lightbox, carrusel, formulario y WhatsApp.
- `assets/images`: imagenes optimizadas para web.
